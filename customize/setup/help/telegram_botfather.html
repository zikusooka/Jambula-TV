How to get a Telegram API Bot ID using Botfather

1: Please install Telegram app on your phone.

2: Signup for Telegram

3: Start a new message and search for the recipient named @Botfather:

4: Click on RESTART

5: In message field, type: /newbot

6: Choose a name for your bot. We recommend: JambulaTV

7: Now choose a unique username end with the word "Bot": jtv001Bot

8: Copy the API token. This is the Telegram API Bot asked for during setup
