Please have following mandatory information ready before proceeding with setup:

Internet & LAN settings

  • 3G/4G Dongle (Rarely used these days):

    APN - Assigned by ISP

  • 3G/4G MiFi (Typical setup):

    SSID Name
    WiFi password
    MiFi router web portal admin username and password

  • Local Area Network (static):

    IP address
    Gateway
    DNS servers

  • WiFi MAC Addresses:

    MAC addresses of your Mobile WiFi devices like phones, tablets, laptops etc

Live TV

  • Ask support for your TV tuner type - There are only 2 types now:

    Silicon Labs Si2168
    Sony CXD2820R

Online Services

  • Please sign up for Trakt.TV and create a watch list called JambulaTV

    Trakt Username

  • Create a Telegram API Bot ID, see the instructions at: Telegram Botfather

    Telegram API Bot ID
    Telegram API Chat ID
    Telegram API Username

  • Create a separate and dedicated GMail account that will be used by JambulaTV to send email notifications to you

    GMail Username

    • IMPORTANT:Please visit account settings and enable less secure access, otherwise Google will block your system from sending out emails

  • Create a Google Youtube API account and get these credentials

    Youtube API Key
    Youtube Client ID
    Youtube Client Secret

  • Signup for the following weather services:
    Weather Underground     and Open Weather

    Weather Underground API Key
    Open WeatherMap API Key

Location (Optional)

  • Get your location's Geo-Coordinates. This data is used only to get precise and reliable weather predictions for you

    Latitude (DD)
    Longitude (DD)

Text to Speech

  • Signup for a free Google TTS account

    Google TTS API Key
    Google TTS API ID (if applicable)
